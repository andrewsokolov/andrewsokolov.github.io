Q: Why MailBar request access to contacts?

A: MailBar need access to contacts for show you contacts picture in messages list. If you denied it, you always can enable it in OS X preferences.

Q: Why MailBar request access to calendar?

A: MailBar need access to calendar for show calendar event attached to message in signle message view. If you denied it, you always can enable it in OS X preferences.

Q: How many times I can activate my licence?

A: You can activate your licence 3 times and use MailBar on different devices.

Q: Do I can use MailBar standalone without Mail.app?

A: No. MailBar working only with Mail.app.

Q: I update OS X to new version and get the message:
"MailBarPlugin.mailbundle not compatible with new version of Mail.app".
What I should do?

A: Just restart MailBar.app. If it still not work - contact us.

Q: How I can remove MailBar plugin for Mail.app?

A: Open Utilities -> Terminal.app and run this command:

rm -rf ~/Library/Mail/Bundles/MailBarPlugin.mailbundle
